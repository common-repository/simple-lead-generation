== Description ==

Really Simple WordPress Lead Capture Plugin. While editing post, you can add form with this plugin to capture information. For pro version more field is supported and layout can be choosing.

== Installation ==

1. Upload `LeadGenerationPlugin` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Lead Generation' menu in WordPress
1. Create new post and click rSimplePLX button to create new lead generation.
